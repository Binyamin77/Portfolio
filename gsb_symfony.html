Projet GSB - Devellopé avec le Framwork Symfony

Contexte

Le laboratoire GSB - souhaite informatiser ses données afin d'avoir une vision claire des informations de ses colaborateurs, de ses médicaments, des laboratoires affiliés.

Mission

Réaliser une application qui pourrait gérer les visiteurs, les secteurs différents laboratoires que couvre GSB. Par exemple l'utilisateur arrive sur la page d'acueil, il peut choisir de lister, d'ajouter ou de supprimer les visiteurs présents, les laboratoires, les régions et les secteurs disponibles, afin d'optimiser ses données.

Technologies utilisées

  • PHP (Symfony)
  • HTML
  • CSS
  • Base de données

Compétences acquises

Lors de ce Projet Personnel Encadré, j'ai pu apprendre de nombreuses fonctionnalités avec le Framwork Symfony que je n'avais pas abordé lors de mon stage de fin d'année dernière.
