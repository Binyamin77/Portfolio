Projet GSB - Java







Contexte

Le laboratoire GSB - Galaxy Swiss Bourdin souhaite informatiser ses données afin d'avoir une vision claire des informations de ses collaborateurs, de ses médicaments, des laboratoires affiliés.

Missions

Réaliser une application qui pourrait gérer les visiteurs, les secteurs et les différents laboratoires que couvre le groupe GSB. Par exemple, l'utilisateur arrive sur la page d'acueil, il peut choisir de lister, d'ajouter, de modifier les informations ou de supprimer les visiteurs enregistrés en base de données, les laboratoires, les régions et les secteurs disponibles afin d'optimiser ses données.

Technologies utilisées

  • Java
  • Swing

Compétences acquises

Lors de ce projet partagé, j'ai pu approfondir mes connaissances sur le langage Java, notamment grâce aux interfaces graphiques Swing.
