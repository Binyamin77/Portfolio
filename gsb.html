Projet GSB - Devellopé avec Java

Contexte

Le laboratoire GSB - Galaxy Swiss Bourdin - souhaite informatiser ses données afin d'avoir une vision claire de ses données.

Mission

L'idée serait de réaliser une application qui pourrait gérer visiteurs, les secteurs différents laboratoires que couvre Galaxy Swiss Bourdin. Par exemple l'utilisateur arrive sur la page d'acueil, il peut choisir de lister, d'ajouter ou de supprimer les visiteurs présents, les laboratoires, les régions et les secteurs disponibles, afin d'optimiser ses données.

Compétences acquises

Lors de ce projet partagé, j'ai pu approfondir mes connaissances sur le langage Java, notamment grâce aux interfaces graphiques Swing.
